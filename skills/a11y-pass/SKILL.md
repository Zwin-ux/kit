# Instructions

1. Identify interactive screens or components in scope.
2. Check labels, roles, keyboard reachability, and focus order.
3. Check contrast for text and critical controls.
4. Ensure errors and empty states are announced in plain language.
5. Fix the highest-impact issues first.
6. Avoid visual redesigns that do not improve access.
7. Summarize before/after access improvements.
