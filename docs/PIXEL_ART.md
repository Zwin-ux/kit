# Pixel Art Direction for Kit

## Mascot
**Kit** — a small, confident baby fox (kit fox) holding a wrench.

Personality: helpful, a little mischievous, competent, never corporate.

## Style Rules
- True pixel art (no high-res anti-aliased illustrations)
- Limited palette (suggested base):
  - Fox orange / amber
  - Cream / off-white
  - Deep navy / blue background
  - Black for outlines and eyes
  - One bright accent (mint or soft yellow) for magic / success / energy
- Clean silhouettes that read at small sizes (TUI icons, 16×16, 32×32)
- Consistent outline weight
- Subtle idle animations later (tail flick, wrench gleam, blink)

## Usage
- GitHub avatar & social
- TUI splash screen
- Loading / success / empty states
- Skill card decorations
- Possible future “Kit reacts” moments in the TUI

## Asset Pipeline (later)
- Source in Aseprite or similar
- Export at multiple sizes
- Keep a living style guide in this folder
