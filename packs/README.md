# Starter Packs

Official curated skill packs for Kit.

| Pack | Use for |
|------|---------|
| `essentials` | Any project (default starter) |
| `web-app` | Apps and sites |
| `library` | Libraries and SDKs |

See [docs/STARTER_PACKS.md](../docs/STARTER_PACKS.md).

```sh
kit pack list
kit pack show essentials
kit pack install essentials
kit pack apply essentials --dir .
```
